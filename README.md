# Rewards-Store
A single page app to "buy" your favorite products

## Tools
- CRA (Create React App)
- FontAwesome
- React.js - Moment.js
- React Router & React Redux
- Github Pages

## Installation
- Install dependencies:
  `$ npm install `
- Run the project with NPM:
 ` $ npm start`
- Find a Demo in https://zule10.github.io/rewards-store
